# Food_Donation_Project
Food Donation project

## Prerequisites

- [GitHub account]
- [Azure account]
- [Azure CLI]
- [.NET Core 3.1 LTS]
- [Azure Functions Core Tools]
- [Visual Studio Code]
- [Visual Code Extensions]
  - [Azure Functions Extensions]
   - [Azure Storage Extension]
